# Результаты тестирования FVSC

## T4: Сходимость recursive deepening (2026-03-29)

**Параметры:** 50 синтетических суждений, 15 понятий, dim=50.

| alpha | Сходимость (k) | Δeig финальное | Δcont финальное | Рост массы |
|-------|----------------|----------------|-----------------|------------|
| 0.5   | k=9            | 9.7e-7         | 2.8e-7          | 0.992x     |
| 0.6   | k=7            | 4.5e-7         | 4.7e-7          | 1.007x     |
| 0.7   | k=6            | 4.9e-7         | 4.2e-7          | 1.006x     |
| 0.8   | k=5            | 2.3e-7         | 2.2e-7          | 1.001x     |
| 0.9   | k=4            | 2.2e-7         | 1.8e-7          | 1.000x     |

**Критерий сходимости:** Δeig < 1e-6 AND Δcont < 1e-5.

**Выводы:**
- Все alpha сходятся. Более высокие alpha сходятся быстрее (меньше итераций), но дают меньше обогащения от содержимого.
- alpha=0.7 (текущее) — хороший баланс: сходимость за 6 итераций, массо-стабильно (1.006x).
- alpha=0.8 — рекомендация для крупных пространств: сходимость за 5 итераций, минимальный рост массы.
- Рост массы < 1% для всех alpha — рекурсивное углубление не раздувает карту.
- **Early stopping на k=4–5 при alpha=0.8 безопасен.** На k=3 при alpha=0.7 тоже (финальное Δeig уже ~1.5e-4).

**Паттерны ошибок:**

| Тип ошибки | Примеры | Возможное решение |
|------------|---------|-------------------|
| Лемматизация spaCy (люди→человек) | "Искренность привлекает людей" → `человек` вместо `люди` | Нормализовать леммы при сравнении |
| Пассивная инверсия неточна | "Дружба основана на доверии" → `доверие→основать→дружба` | Улучшить passive detection для краткого причастия |
| Непереходные глаголы (без объекта) | "Надежда помогает выстоять" — `выстоять` не извлечён как объект | Расширить fallback на xcomp с pos_=VERB |
| Модальный + именной предикат | "Человек должен быть свободен" — не извлечён | Обработать должен+быть+ADJ как S→cop:adj |
| Short adjective predicate | "Свобода важна" — `важна` не распознан | Обработать ADJ-предикаты как copula |

**Корректно обработанные конструкции:**
- Координация (S1 и S2 → V → O): OK
- Отрицание: OK
- Копула (X — это Y): OK
- Условные (если...то): OK (правильно фильтрованы)
- Эпизодические (вчера, этот): OK (правильно отфильтрованы)
- Цитаты (Кант говорил): OK
- Модальные оболочки (верить что): OK
